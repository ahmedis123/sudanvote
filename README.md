# SudanVote

هو تطبيق مبتكر يهدف إلى تبسيط عملية التصويت ومتابعة نتائج الانتخابات بشكل مباشر وشفاف. تم تصميم التطبيق ليكون الوسيلة الأسهل والأكثر أمانًا للمواطنين للمشاركة في العملية الديمقراطية، سواءً عبر التصويت الإلكتروني أو متابعة النتائج الفورية.

## الميزات

- **تصويت إلكتروني:** يتيح للمستخدمين الإدلاء بأصواتهم بطريقة سهلة وآمنة.
- **متابعة النتائج:** تحديث فوري لنتائج الانتخابات لعرض البيانات بشكل شفاف.
- **تطبيق متعدد المنصات:** يعمل على Android, iOS, Web, Linux, macOS, و Windows.
- **واجهة مستخدم جذابة وسهلة الاستخدام.**

## Getting Started

This project is a starting point for a Flutter application, providing a cross-platform solution for electronic voting and election monitoring.

### المتطلبات

- [Flutter SDK](https://flutter.dev)
- بيئة تطوير متكاملة مثل Android Studio أو VS Code
- أجهزة أو محاكيات للتجربة على الأنظمة المستهدفة (Android, iOS, إلخ)

### التثبيت

1. **استنساخ المستودع:**
   ```bash
   git clone https://github.com/ahmedis123/sudanvote.git
   ```
2. **الدخول إلى مجلد المشروع:**
   ```bash
   cd sudanvote
   ```
3. **تثبيت الحزم المطلوبة:**
   ```bash
   flutter pub get
   ```

### التشغيل

لتشغيل التطبيق على جهازك أو المحاكي، استخدم الأمر التالي:
```bash
flutter run
```
يمكنك أيضًا تشغيل التطبيق على منصات متعددة باستخدام الأدوات الخاصة بكل نظام.
